A modern, fully responsive landing page designed for a medical weight loss center. This project focuses on user conversion through a clean interface, interactive elements, and a strong emphasis on accessibility.

## üöÄ Key Features

* **100% Responsive Design:**
    * **Desktop (1920px+):** Spacious layout with 4-column grids and side-by-side alignment.
    * **Tablet (1024px):** Custom hybrid layout (Grid + Flexbox) featuring overlapping elements and optimized spacing.
    * **Mobile:** Vertical stacked layout (Mobile First approach) optimized for readability and touch navigation.
* **Accessibility (A11Y):**
    * Includes a **Colorblind/High Contrast Mode** toggled via a floating button, switching the color palette to Blue/Yellow for better visibility.
* **Interactive Elements:**
    * **Hamburger Menu:** Animated with smooth transitions for mobile devices.
    * **Countdown Timer:** JavaScript-based logic for limited-time offers.
    * **FAQ Section:** Interactive accordion for frequently asked questions.
    * **"Back to Top" Button:** Appears upon scrolling to enhance navigation.
* **Advanced Styling:** Built using CSS Variables (`:root`), CSS Grid, Flexbox, and pseudo-elements for background decorations.
